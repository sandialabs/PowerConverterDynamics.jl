# Examples

The package docs now include a fully executed literate walkthrough that simulates converter, bus, and battery dynamics and generates plots.

- See [Literate Walkthrough](generated/converter_battery_dynamics.md).

That example covers:
- wild source voltage mapped to a common bus,
- converter current/power limits and losses,
- bus voltage behavior under current deficit,
- battery charge/discharge support and SOC evolution,
- plotted outputs for voltage, currents, powers, losses, and efficiency.
